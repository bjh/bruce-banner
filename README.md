## Installation

Add this line to your application's Gemfile:

    gem 'bruce-banner'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bruce-banner

## Usage

basic usage `bb(thing)`
advanced `bb(thing, sentinal:'-', count:20)`
